Frankenstein has a nice sintax, and supports a lot of things. Variable Bindings? Yes! High–Order functions? Yes! Closures? Yes! Data Types? Course yes!

Which features does Frankenstein have?

  • Frankenstein supports function literals and we can use them to bind a function to a name.

  • Data types like booleans, strings, hashes, integers, and arrays.

  • Functions are first–class citizen, treated like any other value. Use high–order functions and pass functions around as values.

  • And, of course, closures.
